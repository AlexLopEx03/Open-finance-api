# Contribution Guide

#### The project aims to improve API and expand its features, including:

- More query parameters and customization options for API responses.

- Completion of the /crypto and /bets endpoints.

- Feedback to improve existing features and its fixing proposals.

- Preventing of abuse and IP blocking mechanisms.

- Better caching and performance optimizations.

---

### If you are still interested to contributing, please follow these steps:

- 1️⃣ Discuss first
>
> Open a discussion in the [Discussions section](https://github.com/AlexLopEx03/Open-finance-api/discussions) to propose your idea or feature, once the proposal is reviewed and approved, you can move to the next step.

- 2️⃣ Fork and implement
>
> Fork the repository's development branch and implement your changes in a dedicated branch.
>
> Ensure the changes follow the project standards and best practices.


- 3️⃣ Submit a Pull Request
>
> Open a Pull Request with your changes, including a brief description of what was implemented and referencing the discussion if applicable.
